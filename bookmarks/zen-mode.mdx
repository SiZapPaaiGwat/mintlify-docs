## 进入 Zen Mode

在书签列表中点击全屏图标，即可进入 Zen Mode。进入 Zen Mode 后，当前屏幕仅展示一条书签，并隐藏左侧导航栏。这样可以专注于当前阅读的书签，提高效率。

## 退出 Zen Mode

按下 Esc 键即可退出 Zen Mode。

## Customize Font Size

进入 Zen Mode 后，点击右上角的图标，自由拖动字体大小调节器即可。

## Customize Font Family

进入 Zen Mode 后，点击右上角的图标，找到您想要的字体，点击 Apply 按钮即可切换字体。
目前仅提供部分字体供选择。
